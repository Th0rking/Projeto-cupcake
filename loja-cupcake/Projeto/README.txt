# Loja do Cupcake

Este projeto é o Sistema de Loja de Cupcake desenvolvido para o Projeto Integrador Transdisciplinar. Ele contempla cadastro e login de usuários, catálogo de cupcakes, montagem personalizada, carrinho, pedidos, painel administrativo e relatórios.

## Estrutura do Projeto

- `/docs`: documentação e diagramas
- `/database`: scripts do banco de dados
- `/backend`: código backend (Python Flask)
- `/frontend`: código frontend (HTML, CSS, JS)
- `/tests`: testes automatizados

## Como rodar

1. Criar e ativar ambiente virtual Python
2. Instalar dependências com `pip install -r requirements.txt`
3. Configurar `.env` com variáveis do banco e chaves de segurança
4. Rodar backend com `python backend/app.py`
5. Abrir frontend localmente no browser

## Testes

Executar `pytest` na pasta `/tests`
